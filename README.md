# Baitong Ticket

Baitong Ticket is a simple ticket booking web application built with React and TypeScript.
This project was created to practice frontend development, Firebase integration, and basic booking system workflow.

# Features

User registration and login (Firebase Authentication)

Event listing page

Ticket selection

Add to cart

Order confirmation

QR code payment simulation (PromptPay)

Order data stored in Firebase Firestore

# Tech Stack

React

TypeScript

React Router

Tailwind CSS

Firebase (Authentication, Firestore)

# What I Learned

How to structure a React project with reusable components

Managing authentication and user data with Firebase

Handling booking and order logic

Generating QR code for payment simulation

Designing responsive UI with Tailwind CSS

How to Run
npm install
npm run dev


Firebase configuration is required.
You need to create your own Firebase project and update the config file.

# Note

This project is built for learning purposes and simulates payment flow using QR code without real transaction processing.
